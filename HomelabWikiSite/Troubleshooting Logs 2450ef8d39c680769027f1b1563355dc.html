Issue Description

When trying to access the Immich web GUI via the IP address, it cannot find the server. The immich web GUI can be accessed using localhost:2283 but not any of the given ip addresses.

Steps to Reproduce

  1. Connect to local network on device
  1. Open Browser or Immich app
  1. Type in 172.18.0.4:2283

Expected Behavior

According to the docker container config, the above ip address should connect to the server

Actual Behavior

Devices and even the localhost PC cannot find the immich server and gives error: Server is not reachable.
