Infrastructure Overview

Homelab Infrastructure Overview

Physical Components

  • Main Server: Dell PowerEdge R520 (16GB RAM, Xeon E5-407 @ 2.4GHz)
  • Main PC: TUF X570-PLUS WIFI (32GB RAM, Ryzen 7 5800X 8 Core 3.8GHz, Radeon RX 7800 XT)
  • Storage Server: Custom ZFS with 3 x 4TB, 2 x 1TB, 1 x 2TB, 1 x 16TB drives in ZRAID 2
  • Network: Microtik Router, TP-Link TL-SG1024 24 port Gigabit Switch, Netgear Nighthawk Wireless Access Point, Netgear Prosafe GS108 8 port Gigabit Switch
  • Power: Startech 8 outlets PDU 125V 15A, surge protectors

Software Infrastructure

  • Hypervisor: Proxmox VE for virtualization
  • Containers: Docker with Portainer and Docker Compose
  • Storage: Proxmox ZFS pools, NFS/SMB shares
  • Networking: Pi-hole for DNS/ad-blocking, pfSense firewall with IDS/IPS

Self-hosted Applications

  • Media: Plex, Jellyfin
  • Home Automation: Home Assistant
  • Monitoring: Grafana, Prometheus, InfluxDB
  • Documentation: Wiki.js
  • Password Management: Vaultwarden
  • Git Server: Gitea

Network Topology

Security Implementation

  • Network Segmentation: Multiple VLANs for IoT, guest, and main networks
  • Intrusion Detection: Suricata IDS/IPS on pfSense
  • Traffic Monitoring: Netflow analysis with ELK stack
  • Access Control: Strong authentication with MFA where available

Backup Strategy

  • Local: Daily incremental backups to NAS
  • Offsite: Weekly backups to cloud storage (Backblaze B2)
  • Configuration: All network and server configs version controlled
  • Recovery: Documented procedures for rebuilding services

Future Expansion Plans

  • Redundant internet connection with automatic failover
  • Storage capacity expansion with additional drives
  • Network upgrade to 10Gbe for server interconnects
  • Dedicated hardware for machine learning projects
